2.6.3 Make Your Own Requests

  • /attackTheDarkness -- user supplies a name and weapon with which to attack the darkness. Example: /attackTheDarkness?name=Batman&weapon=Batfist
  • /badGambler -- user gets 2 guesses at a random number. Example: /badGambler?num1=5&num2=87
